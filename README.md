# Модуль управления сочлененной робо-рукой (Flask + UDP + SQLite)

Локальный веб-сервер на Python и Flask для координационного управления многопальцевой робо-рукой через сетевые UDP-запросы. Программная логика и интерфейс разработаны специально для физической модели, созданной по моему индивидуальному чертежу (механика на базе гибких тросов и возвратных пружин).

## Описание работы системы
Интерфейс представляет собой пульт управления. Благодаря интеграции скрытого фрейма (`iframe`), отправка сетевых команд происходит бесшовно в фоновом режиме, без визуального обновления или мерцания веб-страницы.

* **Команда ON (Включение)**: Активация общего питания сервоприводов конструкции, натяжение управляющих тросов (пальцы переводятся в рабочее положение).
* **Команда OFF (Выключение)**: Безопасное отключение питания моторов. Кабели расслабляются, и под воздействием механических возвратных пружин пальцы автоматически разжимаются в исходную позицию.

Все действия и команды фиксируются в базе данных SQLite (`logs.db`) с сохранением точного времени события. Посмотреть накопленный журнал можно по адресу: `http://127.0.0`

## Системные требования
* Python 3.x
* Библиотека Flask (установка: `pip install flask`)

## Запуск проекта
1. Запустите Flask-сервер из терминала:
   ```bash
   python server.py
   ```
2. Откройте панель управления в браузере: `http://127.0.0`

## Архитектура файлов проекта
* `server.py` — основной файл бэкенда (сервер Flask, обработка маршрутов `/on`, `/off`, `/logs` и отправка UDP на робота).
* `logs.db` — структурированная база данных истории команд (создается автоматически).
* `robohand.ino` — микропрограмма на C++ для микроконтроллера Arduino, отвечающая за прием сетевых пакетов и физическое движение моторов.
* `README.md` — данная техническая документация (этот файл).
