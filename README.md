# Arduino UDP Controller — Панель управления рукой робота

Простой веб-сервер на Python + Flask, который позволяет отправлять UDP-команды на Arduino для управления рукой робота («Включить»/«Выключить» или «Открыть»/«Закрыть») через веб-интерфейс. Каждое действие логируется в базу данных SQLite с отметкой времени.

## Описание
Сервер запускается локально. С помощью кнопок можно управлять состоянием руки робота. При нажатии на кнопку браузер отправляет запрос по адресу /on или /off. Flask-сервер перехватывает эти запросы, отправляет UDP-пакет на Arduino, делает запись в базу данных logs.db и возвращает текстовый ответ "OK".

Этот ответ "OK" загружается внутрь скрытого фрейма iframe на странице. Для пользователя всё выглядит бесшовно: он нажимает на кнопку, пульт остается на экране без перезагрузки страницы, а команда мгновенно уходит на Arduino.

Все события записываются в базу данных logs.db и накапливаются. Посмотреть историю действий можно по адресу: http://127.0.0

## Требования
* Python 3.x
* Flask

## Установка и запуск
1. Установите Flask (если не установлен):
     pip install flask
   2. Запустите сервер:
     python server.py
   3. Откройте интерфейс в браузере: http://127.0.0

## Маршруты (Адреса)
* http://127.0.0 — Главная страница управления (пульт)
* http://127.0.0on — Отправить UDP-команду ON и записать в БД
* http://127.0.0off — Отправить UDP-команду OFF и записать в БД
* http://127.0.0 — Страница со списком всех логов из базы данных
